# CNN-LSTM Weather Prediction

## Project Overview
This project uses a hybrid CNN-LSTM deep learning model to predict weather patterns using time-series data.

## Model Architecture
- CNN layers for feature extraction
- LSTM for sequential learning
- Dense layers for final prediction

## Dataset
Stored in Google Drive (large dataset):
https://drive.google.com/drive/folders/138PGI3jgCBqzoGs2q70h0kPDBKm2q2q-?usp=drive_link

## Tech Stack
- Python
- TensorFlow / Keras
- Pandas, NumPy
- Scikit-learn

## Workflow
1. Data preprocessing
2. Scaling (MinMaxScaler)
3. Sliding window creation
4. CNN-LSTM model training
5. Prediction

## Result
Model trained for weather forecasting using deep learning.
